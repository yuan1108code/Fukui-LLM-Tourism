## 🔧 進階設定

### 1. 啟用完整 AI 功能
如果您想使用真正的 OpenAI GPT 和 ChromaDB：

1. 確認 `.env` 檔案中的 `OPENAI_API_KEY` 設定
2. 停止當前測試服務
3. 啟動完整版後端：
```bash
cd backend
python3 app.py
```

### 2. 設定 Mapbox 地圖
為了獲得更好的地圖體驗：

1. 註冊 [Mapbox 帳號](https://www.mapbox.com/)
2. 取得 Access Token
3. 在 `frontend/.env` 中設定：
```bash
VITE_MAPBOX_ACCESS_TOKEN=your_actual_mapbox_token
```
